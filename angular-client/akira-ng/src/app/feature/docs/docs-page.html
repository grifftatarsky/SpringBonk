Case study

Bonk! ranked-choice book club

The Bonk! Book Club uses Akira to run preferential elections every reading cycle:

  • Each meeting opens with nominations seeded from shelves or custom picks so the ballot mirrors what the club is reading right now.
  • Members rank as many titles as they want using American ranked-choice rules, and the election detail page surfaces every instant-runoff redistribution.
  • When the organizer taps “Close election,” the workflow locks the slate, announces the winning book, and queues the next reading cycle.

Because every step is transparent, members see how their preferences roll up into the final outcome and why the winning book fits the club’s taste profile.
